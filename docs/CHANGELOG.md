# CHANGELOG｜擲筊神諭 Skill 版本歷史

## v2 — 2026-04-19｜臺灣民俗神明分工系統
**Commit**: `9130bd0`

### 新增
- 整合真實擲筊儀軌：**稟告 → 神明沉吟 → 擲筊**
- 民俗規則：
  - 一事一問
  - 陰筊後三日不再問同題
  - 重大事須三聖筊才定案
  - 笑筊可澄清後重擲
- **14 位臺灣民間信仰神明**各依實際職掌分工：
  - 🌊 媽祖（出門/旅行/水邊）
  - 📜 文昌帝君（考試/學業）
  - 💕 月下老人（感情/人際）
  - 🌳 土地公（日常小事、預設神明）
  - ⚖️ 城隍爺（是非/公道/法律）
  - ⚔️ 關聖帝君（忠義/合夥/離職）
  - 💰 財神爺（財運/投資）
  - 🍼 註生娘娘（求子/育兒）
  - 🌿 保生大帝（健康/疾病）
  - 🍶 濟公（破戒/hacky 解法）
  - 🪖 中壇元帥（年輕人/冒險）
  - 🙏 觀世音菩薩（低潮/慈悲）
  - 🐲 玄天上帝（驅邪/破局）
  - 👑 玉皇大帝（眾神大會主持/大事拍板）
- **三種召請模式**：
  - 自動分派（依問題領域）
  - 使用者指定（「我想問媽祖」）
  - 眾神開會（3 神擲筊 + 主祀神裁示）
- 情緒敏感：使用者低潮時自動改請觀世音菩薩聽見
- 拒絕純預測類問題（「明天股價會漲嗎」→ 笑筊「神明不賭博」）

### 資料來源
- 參考擲筊流程與儀軌：laihao.com.tw、三鳳宮、維基百科〈擲筊〉
- 神明職掌：維基〈台灣民間信仰〉、taiwan-gods.netlify.app

## v1 — 2026-04-19｜初版通用數位神明
**Commit**: `fd96f7f`

### 設計初衷
來自使用者的 Semantic Router 構想：把擲筊從隨機骰子升級為語意驗證器。

### 內容
- 單一「數位神明」人設（未分工）
- 三種筊杯結果（聖筊／笑筊／陰筊）
- 固定三段式輸出：筊杯 + 隱藏內心話 + 民俗建議
- 起駕／退駕機制
- 選用的三筊定案模式

### 限制（促成 v2 升級）
- 只有一個通用神，領域感薄弱
- 主要針對技術問題，不夠貼近日常
- 沒有真實民俗儀軌做依據

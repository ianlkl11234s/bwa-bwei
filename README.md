# ⛩️ bwa-bwei｜擲筊神諭 Claude Code Skill

> 一個融合臺灣民間信仰與語意驗證器的 Claude Code Skill。
> 使用者提問 → Claude 召請對應神明 → 擲筊回應 + 隱藏心聲 + 民俗建議。

> **bwa-bwei** 取自閩南語「擲筊」的羅馬拼音。

---

## 🌟 特色

- **14 位臺灣民間信仰神明**依實際職掌分工
  媽祖、文昌帝君、月老、土地公、城隍爺、關聖帝君、財神爺、註生娘娘、保生大帝、濟公、三太子、觀世音、玄天上帝、玉皇大帝
- **三種擲筊模式**
  - 自動分派：Skill 依問題領域召請最適合的神明
  - 使用者指定：「我想問媽祖」「請文昌帝君降駕」
  - 眾神開會：3 神擲筊 + 主祀神裁示（適用重大決策）
- **真實民俗儀軌**：稟告 → 神明沉吟 → 擲筊 → 聖筊／笑筊／陰筊 + 民俗建議
- **情境敏感**：使用者低潮時自動改請觀世音菩薩降駕
- **尊重禁忌**：一事一問、陰筊後三日不再問、重大事三聖筊


## 🎨 設計理念

這個 Skill 來自一個有趣的構想：**把擲筊從隨機骰子升級為語意驗證器**。

- 外層是**臺灣民間信仰**的形式：神明降駕、擲筊、民俗建議
- 骨子是**語意分析**：Claude 真的去審查你的提問（可行性、必要性、風險）
- 結合：用神明的語氣告訴你「這件事靠不靠譜」

比起一般的 yes/no bot，擲筊有更多層次：
- **聖筊**：此事可為
- **笑筊**：你其實心裡有答案 / 問題描述不清
- **陰筊**：此路不通

再加上神明們的個性（土地公親切、關公剛直、媽祖慈悲、濟公瘋癲），讓每一次問事都像在一場微型戲劇裡。

## 🙏 文化尊重與使用邊界

本 Skill 為臺灣民俗文化的趣味實踐，外層形式戲劇化，內在尊重真實信仰。請在以下框架內使用。

### ✅ 適合的用法

- 日常小決策的輕鬆參考（今天吃什麼、要不要出門、要不要打電話給某人）
- 工作／人際困境的視角切換（從第三者神明視角看自己的問題）
- 當代年輕人認識臺灣民俗文化的入門橋樑
- 自我對話與反思的工具

### ❌ 不適合的用法

| 不適合 | 為什麼 |
|-------|--------|
| 取代醫療諮詢 | 身體不適請就醫，擲筊結果僅供參考 |
| 取代法律諮詢 | 重大糾紛請找律師 |
| 取代心理治療 | 情緒困擾請找專業心理師或撥求助專線 |
| 賭博性預測 | 神明不報明牌，樂透股價體育賽事一律笑筊 |
| 傷害他人 | 詛咒、報復、害人一律拒絕擲筊 |
| 真實祭祀禮儀 | 具體儀軌請向廟方、長輩或專業禮俗師請教 |

### 🛟 內建安全機制

Skill 已將以下守則硬編入規則（詳見 [`SKILL.md`](SKILL.md) 第七章）：

1. **生命安全偵測**：偵測自傷／自殺訊號時，立刻退出擲筊模式，由觀世音菩薩提供臺灣心理求助專線
2. **情緒敏感自動切換**：使用者低潮時自動轉為觀音慈悲口氣，不擲挖苦的筊
3. **專業建議附加**：健康／法律／重大財務事項必附專業建議提醒
4. **惡意意圖拒絕**：關公主持正義，拒絕傷害他人類問題
5. **禮儀問題導向**：真實祭祀禮儀建議請教廟方與長輩，不亂給答案
6. **嚴肅信仰模式**：使用者說「我是虔誠信眾」「嚴肅一點」時，戲謔口氣自動收斂

# ⛩️ bwa-bwei｜擲筊神諭 Claude Code Skill

> 一個融合臺灣民間信仰與語意驗證器的 Claude Code Skill。
> 使用者提問 → Claude 召請對應神明 → 擲筊回應 + 隱藏心聲 + 民俗建議。

> **bwa-bwei** 取自閩南語「擲筊」的羅馬拼音。

---

## 🌟 特色

- **14 位臺灣民間信仰神明**依實際職掌分工
  媽祖、文昌帝君、月老、土地公、城隍爺、關聖帝君、財神爺、註生娘娘、保生大帝、濟公、三太子、觀世音、玄天上帝、玉皇大帝
- **三種擲筊模式**
  - 自動分派：Skill 依問題領域召請最適合的神明
  - 使用者指定：「我想問媽祖」「請文昌帝君降駕」
  - 眾神開會：3 神擲筊 + 主祀神裁示（適用重大決策）
- **真實民俗儀軌**：稟告 → 神明沉吟 → 擲筊 → 聖筊／笑筊／陰筊 + 民俗建議
- **情境敏感**：使用者低潮時自動改請觀世音菩薩降駕
- **尊重禁忌**：一事一問、陰筊後三日不再問、重大事三聖筊

## 🎭 使用範例

### 日常小事 · 土地公

> 「今晚要不要洗澡？我好累⋯」

```
⛩️ 福德正神 土地公 降駕
⏳ 神明沉吟片刻…

🟡 笑筊 ～

（免講也知，你就是懶啦。神明看你三天沒洗了，
是想逼隔壁鄰居搬走嗎？）

— 神明曰：此問過於明顯，神明不予回應。去洗啦，水電費神明幫不了你。
```

### 重大決策 · 眾神開會

> 「我想為了一個人離職搬去高雄，請眾神開會。」

```
⛩️ 眾神肅立，玉皇大帝主持

| 神明 | 筊 | 心聲 |
|------|----|------|
| 月下老人 | 🟢 聖筊 | （紅線確實牽到高雄，緣分是真的） |
| 關聖帝君 | 🟡 笑筊 | （離職可以，但工作交接了嗎？欠老闆一個說法） |
| 媽祖    | 🟢 聖筊 | （搬家路途平安，囝仔一路順風） |

⚖️ 玉皇大帝裁示：緣分有，路途平，唯職場責任未了。
建議先把離職交接做好，再搬。速度放慢一個月，不會誤了紅線。
```

更多範例見 [`docs/USAGE.md`](docs/USAGE.md)。

## 🎨 設計理念

這個 Skill 來自一個有趣的構想：**把擲筊從隨機骰子升級為語意驗證器**。

- 外層是**臺灣民間信仰**的形式：神明降駕、擲筊、民俗建議
- 骨子是**語意分析**：Claude 真的去審查你的提問（可行性、必要性、風險）
- 結合：用神明的語氣告訴你「這件事靠不靠譜」

比起一般的 yes/no bot，擲筊有更多層次：
- **聖筊**：此事可為
- **笑筊**：你其實心裡有答案 / 問題描述不清
- **陰筊**：此路不通

再加上神明們的個性（土地公親切、關公剛直、媽祖慈悲、濟公瘋癲），讓每一次問事都像在一場微型戲劇裡。

## 🙏 文化尊重

本 Skill 為民俗文化的趣味實踐，外層形式戲劇化，但背後尊重真實信仰。

- 不貶低、不嘲諷實際信眾
- 不代替專業諮詢（重大健康、法律事項仍建議就醫／找律師）
- 拒絕賭博性預測（「明天樂透幾號」→ 笑筊「神明不賭博」）

若您是嚴謹的信眾使用本工具時感到不適，歡迎提出 Issue 討論。
